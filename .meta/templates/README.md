# Templates Directory

This directory is for **output templates** - reusable formats for consistent results.

---

## What Goes Here

When you need consistent output formats across toolkits, create templates here.

**Examples of what templates might be**:
- Report formats
- Summary structures  
- Email layouts
- Analysis frameworks

---

## Creating Templates

When you need a reusable template:

1. Create `.meta/templates/your-template.template.md`
2. Define sections and structure
3. Add instructions for AI
4. Reference from your toolkit workflows

---

## This Directory is Currently Empty

That's intentional. Create templates when you need consistent outputs.

**Workflow**:
1. Generate toolkit → use it → need consistent format
2. Create template for that format
3. Add to this directory
4. Reference in future toolkits

---

**Templates are optional. Only create when consistency matters.**
